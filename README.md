# Password Generator Pro

A simple and user-friendly Password Generator desktop application developed using " Python " and " Tkinter ".

This application allows users to generate strong and secure passwords with customizable options.



# Features

- Generate secure random passwords
- Adjustable password length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include special symbols
- Copy password to clipboard
- Reset all options
- Clean and modern GUI
- Input validation


# Technologies Used

- Python 3
- Tkinter
- Random Module
- String Module
